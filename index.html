VRPN the Virtual Reality Peripheral Network

The device-independent, network-transparent system for accessing virtual reality peripherals from VR applications.

Open source

The VRPN library is licensed under the Boost Software License 1.0, and has been developed cooperatively by the virtual reality community for well over a decade.

Ready to use

The VRPN protocol has been integrated into a number of hardware devices and VR software platforms. The VRPN library package also includes a server with extensive built-in or optional device support, as well as basic tools for viewing device data.
